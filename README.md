# WEATHER APP (PYTHON + PYQT5)

* A simple desktop weather application built using Python.
* The app allows users to enter a city name and view real-time weather information.

## FEATURES
- Search weather by city name
- Displays temperature and weather condition
- Weather emojis for better visual understanding
- Handles invalid city names and API errors
- Simple and clean desktop interface

## TECH STACK
- Python
- PyQt5
- Requests
- OpenWeather API

## HOW TO USE
1. Go to the Releases section of this repository
2. Download main.exe
3. Double click the file to open the app
4. Enter a city name and click Get Weather

## NOTE
* Windows may show a security warning when opening the app.
* Click "More info" and then "Run anyway" to start the application.
