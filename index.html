Weather App – Python Project

A desktop weather application built using Python and PyQt5

Project Overview

This Weather App allows users to enter a city name and fetch real-time weather information using the OpenWeather API. The application is built as a desktop GUI using PyQt5.

Key Features

  • City-based weather search
  • Real-time temperature and conditions
  • User-friendly PyQt5 interface
  • Error handling for invalid inputs

Tech Stack

  • Python
  • PyQt5
  • Requests
  • OpenWeather API

How to Run

  • Install dependencies using pip install -r requirements.txt
  • Add your OpenWeather API key in the code
  • Run the app using python main.py
